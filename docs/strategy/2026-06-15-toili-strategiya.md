# Тойли (Toili) — Стратегия запуска

**Дата:** 15 июня 2026
**Статус:** утверждено (брейншторм-сессия)
**Продукт:** платформа электронных приглашений с фокусом на Кыргызстан → далее Казахстан и Россия
**Главный конкурент:** Shaqyru24 (shaqyru24.kz) — казахстанский сервис электронных приглашений

---

## 0. Контекст и вводные

- **Ресурсы:** небольшая команда, скромный бюджет на рекламу/интеграции.
- **Стартовые активы:** готовой аудитории/связей пока нет, наращиваем по ходу.
- **Главная ставка GTM:** сильный продукт + вирусность, самостоятельный таргет и UGC-ролики; блогеры — позже, когда появится бюджет.
- **Тайминг:** запуск в текущий сезон тоёв (лето 2026), быстрый MVP без потери качества.

---

## 1. Позиционирование и дифференциация

### Позиция
**«Лучше по качеству — по обычной цене. С душой к нашим традициям.»**

Не премиум по цене и не «дешевле всех» (эту позицию уже занял Shaqyru24). Продаём **заметно более красивый и удобный продукт по цене обычного приглашения**. Гость видит, что приглашение лучше, а стоит как стандартное.

Локальность (кыргызские обряды) — ключевая часть позиции, бьёт в две слабости конкурента: он казахстанский и давит на «дешевизну».

### Дифференциация

| Ось | Shaqyru24 | Тойли |
|---|---|---|
| **Цена** | «самый доступный» | как обычное приглашение + стартовая скидка |
| **Качество/дизайн** | базовое | заметно красивее, анимация, motion |
| **Платежи/подарки** | Kaspi (KZ) | MBANK, O!Деньги, Optima (KG) |
| **Рассылка** | WhatsApp | WhatsApp + Telegram-бот с авто-RSVP (2-я волна) |
| **Шаблоны** | общие | КГ-обряды + универсальные |
| **Карта** | — | 2ГИС + маршрут |
| **Языки** | казахский | кыргызский + русский + казахский |
| **Перевод** | — | авто-перевод (2-я волна) |

### Бренд
- **Название (рабочее):** **Тойли (Toili)** — от «той», тёплое, лёгкое, читается кыргызами и русскими, масштабируется на РФ.
- **Тон:** тёплый, современный, «свой», не корпоративный.
- **Оффер:** «Создай приглашение, которым гордишься — за 5 минут, на трёх языках».
- ⚠️ **До запуска проверить:** свободны ли домены `toili.kg` / `toili.com` и не занят ли товарный знак.

## 2. Монетизация и цены

### Модель: Freemium + оплата за событие
Базовый бесплатный тариф (с бейджем «Сделано в Тойли») работает как **вирусная петля** и воронка. Оплата — разовая за конкретный той, без подписки (люди делают приглашения редко).

### Тарифы (за одно событие)

| Тариф | Что входит | Старт (скидка) | Потом |
|---|---|---|---|
| **Бесплатно** | 5+ базовых шаблонов, RSVP, бейдж «Тойли» | 0 | 0 |
| **Стандарт** | все шаблоны, без бейджа, карта 2ГИС, неогр. гостей | ~390 сом | ~590 сом |
| **Премиум** | анимация/премиум-шаблоны, сбор подарков (MBANK/O!), Telegram-бот, аналитика | ~690 сом | ~990 сом |

*Цифры — гипотеза под рынок КГ, провалидировать по факту. Порядок: «как обычное приглашение».*

### Логика
- **Вводная скидка** первые ~3 месяца («цена раннего доступа») → набор базы и отзывов → потом убираем.
- **Бесплатный тариф = маркетинг**, не убыток: гонит бренд по WhatsApp/Telegram бесплатно.
- **Сбор подарков на старте без комиссии** — фича доверия против Kaspi-завязки конкурента.
- **Авто-конвертация валют** — цены и суммы показываются в валюте пользователя (сом/тенге/рубль) по курсу.
- **Доп. доход (позже):** премиум-шаблоны, видео-приглашения, мини-курсы.

## 3. Go-to-market

Связка: product-led + контент. Бюджет минимальный, поэтому ставка на то, что **продукт сам себя показывает**.

### Рычаги роста (по приоритету)

1. **Вирусная петля (канал №1, бесплатный).**
   Каждое приглашение видят десятки-сотни гостей. На каждом — бейдж «Сделано в Тойли» + кнопка «Создать своё приглашение». Один той = 50-200 показов бренда целевой аудитории бесплатно.

2. **Короткие видео (TikTok / Instagram Reels) — UGC своими силами.**
   Форматы: «приглашение за 5 минут» (экранка), before/after, залипательные анимации под трендовый звук, реакции гостей, «как собрать подарки через MBANK». Цель: 3-5 роликов/неделю, гнаться за насмотренностью, не за продакшеном.

3. **Таргет (Meta: Instagram/Facebook).**
   Усиливать платно уже зашедшие органические ролики. Аудитория: КГ, 20-38 лет, перевес на женщин, интересы «свадьба/события/беременность». Ретаргет посетителей сайта. Старт ~$5-10/день, масштаб по окупаемости.

### Тайминг
Июнь — разгар сезона тоёв, спрос горячий. Pre-launch (1-2 нед, лендинг + ранний доступ + посев TikTok) → Launch (скидка + органика + малый таргет) → Post (отзывы и скрины реакций в контент).

### Партнёрства (этап 2)
Тамада, той-салоны, фотографы, рестораны → приводят клиентов за % или бартер. Масштабный канал, не на старте.

## 4. MVP-скоуп

**Принцип:** «качество в витрину, простота в моторе». Максимум сил — в шаблоны, редактор и страницу готового приглашения. Бэкенд/аналитика/автоматизации — простые.

### ✅ Входит в MVP

| Блок | Что делаем |
|---|---|
| **Галерея шаблонов** | 5+ бесплатных + 3-4 премиум. КГ-обряды (нике, кыз узатуу, бешик той, сүннөт той) + универсальные. ← зона качества |
| **Редактор-конструктор** | короткий мастер: имена, дата, время, место, фото; живой предпросмотр; удобно с телефона. ← зона качества |
| **Готовое приглашение** | красивая страница-ссылка, адаптив, музыка (опц.), анимация. ← зона качества |
| **Рассылка** | ссылка + кнопки «поделиться в WhatsApp / Telegram» + копировать |
| **RSVP** | гость отвечает без регистрации; хозяин видит список «приду / не приду» |
| **Сбор подарков** | ссылка/QR хозяина (MBANK или O!Деньги), без интеграции на MVP |
| **Карта** | ссылка на 2ГИС + «построить маршрут» |
| **Языки** | интерфейс KG + RU + KZ |
| **Авто-конвертация валют** | суммы в валюте пользователя по курсу |
| **Оплата тарифов** | разовая оплата за событие через API MBANK |
| **Вход для хозяина** | по номеру телефона |

### ❌ Не входит в MVP (2-я волна)
Telegram-бот с авто-RSVP, авто-перевод, рассадка гостей, живая стена пожеланий, фото-альбом после тоя, курсы, блог, расширенная аналитика, комиссия с подарков.

### Стек
- **Next.js + Tailwind + shadcn/ui** — быстро и красиво
- **Хостинг: Vercel**
- **БД: Postgres (Neon через Vercel)** — приглашения, RSVP, юзеры

### ⚠️ Риски (проверить рано)
1. **Платежи MBANK** — убедиться, что есть приём оплаты для бизнеса (мерчант/эквайринг/API). Первая техническая задача спринта.
2. **Сбор подарков** на MVP — ссылка/QR хозяина (без интеграции, без юр-рисков).
3. **Анимация шаблонов** — самая трудоёмкая часть качества; начать с 2-3 «вау», расширять.

## 5. Метрики и план на 90 дней

### North Star
**Кол-во опубликованных приглашений в неделю** (отражает ценность + вирусность).

### Воронка
```
Зашли на сайт → создали приглашение → опубликовали → оплатили (платные)
       ↓ гости открыли → нажали «создать своё» (вирусность)
```
Ключевое: конверсия в создание, % оплат, K-фактор (новые юзеры с 1 приглашения), CAC vs цена тарифа.

### План на 90 дней

**Спринт 0 (нед. 1-2) — стройка + проверка**
- 🔴 Проверить MBANK API/эквайринг (риск №1)
- Лендинг + сбор «раннего доступа», посев первых TikTok
- Дизайн 5 бесплатных + 3 премиум шаблона (упор на 2-3 «вау»)
- Каркас: редактор, страница приглашения, RSVP

**Спринт 1 (нед. 3-4) — запуск MVP**
- Релиз со вводной скидкой
- 3-5 роликов/неделю, малый таргет на лучшие
- Первые отзывы и скрины реакций

**Месяц 2 — рост**
- Усиление зашедших роликов платным таргетом
- Доработка популярных шаблонов, новые шаблоны
- Реферальная механика, соцдоказательство на лендинг

**Месяц 3 — закрепление**
- 2-я волна фич: Telegram-бот с авто-RSVP, авто-перевод
- Первые партнёрства (тамада/салоны) на тест
- Решение: убирать ли вводную скидку

### Ориентиры успеха
- Конец мес. 1: запуск + первые 20-50 опубликованных приглашений
- Конец мес. 2: рост неделя к неделе, K-фактор > 0
- Конец мес. 3: окупаемость таргета (CAC < цены премиума), решение по скидке
